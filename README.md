## Installation ğŸ’¾

Clone repository:

```bash
git clone https://github.com:zielvna/portfolio.git
```

Install packages:

```bash
yarn install
```

Build project:

```bash
yarn build
```

Start it:

```bash
yarn start
```

Then go to http://localhost:3000 to see it.
